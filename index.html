Loops

🔄 In programming, we often need to repeat tasks!

For example, we might need to create multiple characters or patterns in games. 🎮

In such cases repeating the same lines of code can become tedious and in programming we believe in the principle of D.R.Y - Do not Repeat Yourself! 🚫🔁

So, in places where you need to repeat the same set of commands, we use loops. 🔁

In Python, there are two types of loops:
• For Loop
• While Loop

In today’s class, we will focus on the for-loop. 🔍

🧮 Understanding the range() function, which is frequently used in conjunction with loops, is important for effectively using loops!

Range Function

Here's the syntax:

range(start, stop, step)

range is a predefined function in Python that helps to create a sequence of numbers. It takes the following parameters:

• start (optional) - An integer number that specifies which number to start. If not specified, the default value is 0.
• stop (required) - An integer number specifying which number to stop.
• step (optional) - An integer number specifying by what value to increment. If not specified, it gets the default value 1.
